■scalac でクラスのパスを指定する方法
--------------------------------------------------------------------------------

Windowsではクラスのパスが複数の場合、区切りとして「;」を使う(Linuxは「:」)
デフォルトの設定を上書きするので、必要ならばカレントのディレクトリ「.」もパスに追加する

[例]
scalac -d classes -classpath .;..\lib\poi\poi-3.12-20150511.jar;..\lib\poi\poi-ooxml-3.12-20150511.jar test_poi.scala

「-d classes」は生成された*.classファイルを吐き出す先の指定


scala SDKのインストール先にある「doc/tools」ディレクトリの中に
あるドキュメントに詳細なオプションの記述がある。
ググるよりも確実かも。
